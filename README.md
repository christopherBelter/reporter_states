# reporter_states
R processing code, resulting data files, and R app for running a Shiny dashboard to display active NICHD award data from RePORTER. 

The `reporter_processing.r` file is used to download, process, and analzye NIH grant data from the RePORTER API using additional helper files located in the "helpers" folder. The .r file generates the summary data files stored in the "data" folder. Finally, the `app.r` file creates the dashboard itself.
